# Course: Building AI Agents for Shopify with MCP

Welcome to the comprehensive course on building next-generation AI shopping assistants for Shopify. This course uses the Shopify Storefront MCP template as its primary case study.

## Syllabus

### [Module 1: Introduction to AI Storefront Agents and MCP](./module_1_intro.md)
- What are AI Agents in commerce?
- Introduction to the Model Context Protocol (MCP).
- Why MCP is a game-changer for Shopify development.

### [Module 2: Architecture and Setup](./module_2_setup.md)
- Understanding the three-tier architecture.
- Environment variables and configuration.
- Getting the app running locally.

### [Module 3: The Backend - LLMs and Tools](./module_3_backend.md)
- Integrating Claude with React Router.
- How the MCP Client works.
- Managing the multi-step conversation loop.
- Implementing Server-Sent Events (SSE) for streaming.

### [Module 4: The Frontend - Building the Widget](./module_4_frontend.md)
- Shopify Theme Extensions explained.
- Liquid, CSS, and Vanilla JS in a React-centric world.
- Handling real-time streams in the browser.

### [Module 5: Security and Customer Data](./module_5_auth.md)
- The PKCE OAuth flow for customer accounts.
- Securely storing and using access tokens.
- Privacy considerations for AI agents.

### [Module 6: Extending the Agent](./module_6_extending.md)
- Modifying the System Prompt.
- Adding custom MCP tools.
- Changing the LLM provider.
